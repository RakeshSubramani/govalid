# GoValid

**GoValid** is a lightweight, powerful, and flexible validation library for Go.  
It provides easy-to-use validators for **strings, numbers, and structs** with support for custom rules and nested struct validations.  

---

## Features

- ✅ String validation (required, length, email, URL, alphanumeric, regex, UUID, etc.)  
- ✅ Number validation (min, max, range, positive/negative, integer/float, multiple of, equality)  
- ✅ Struct validation using `valid` tags, including nested structs and slices  
- ✅ Combine multiple errors into a single error  
- ✅ Extensible and easy-to-use
